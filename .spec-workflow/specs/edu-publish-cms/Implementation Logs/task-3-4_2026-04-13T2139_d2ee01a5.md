### API Endpoints

#### GET /api/auth/github/start
- **Purpose:** Starts GitHub OAuth with PKCE and sets the temporary encrypted OAuth state cookie before redirecting to GitHub.
- **Location:** worker/routes/auth.ts:42
- **Request Format:** No request body.
- **Response Format:** 302 redirect to GitHub authorize URL with Set-Cookie for temporary OAuth state.

#### GET /api/auth/github/callback
- **Purpose:** Validates OAuth state, exchanges the GitHub code for a token, fetches the viewer, and sets the encrypted session cookie.
- **Location:** worker/routes/auth.ts:52
- **Request Format:** Query params: code (string), state (string).
- **Response Format:** 302 redirect to APP_URL on success; JSON error payload on invalid state or upstream auth failure.

#### POST /api/auth/logout
- **Purpose:** Clears the encrypted session and OAuth cookies so the browser returns to an anonymous Worker session.
- **Location:** worker/routes/auth.ts:83
- **Request Format:** No request body.
- **Response Format:** 204 No Content with clearing Set-Cookie headers.

#### GET /api/session
- **Purpose:** Returns the current authenticated GitHub viewer from the Worker session cookie or an anonymous session payload.
- **Location:** worker/routes/session.ts:6
- **Request Format:** No request body.
- **Response Format:** { authenticated: boolean, viewer: GitHubViewer | null }
